LagrangianLM is a production-grade Small Language Model inspired by the LLaMa Architecture

The objective is to understand and implement every component of a modern LLM from first principles

The model will be:
1. ~150 Million Parameters
2. Decoder Only
3. RoPE
4. RMSNorm
5. SwiGLU
6. SentencePiece or BPE Tokenizer
7.PyTorch
8. Pretrained from scratch
9. Instruction tuned
10. Quantized
11. Production grade
